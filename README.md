# Голосовой ассистент

Голосовой ассистент на Python с поддержкой распознавания и синтеза речи. Приложение использует Vosk для распознавания команд и pyttsx3 для озвучивания текстов. Ассистент активируется словом "привет" и поддерживает большой набор команд: таймеры, напоминания, работа с заметками, воспроизведение музыки и видео, игры: в города, математика, викторина, словарь иностранных слов, а также анекдоты, дурацкие советы. Также есть несколько вариантов рандомайзера, такие как бросок кубика, монеты, и обычный рандом в целых числах. Кроме того присутствуют такие функции, как, например, чихающий в рандомный момент ассистент. Также есть функция сна, когда ассистент не реагирует на команды до команды пробуждения.
